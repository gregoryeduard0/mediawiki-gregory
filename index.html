O que é MediaWiki?

O MediaWiki é uma plataforma de software robusta e flexível, projetada para permitir a colaboração online em larga escala. Com sua origem na Wikipedia, o MediaWiki evoluiu para se tornar uma solução confiável para qualquer projeto ou organização que busca um ambiente colaborativo e de gestão de conteúdo eficaz.

Requisitos para Instalar

Além do próprio software, uma instalação padrão do MediaWiki tem os seguintes pré-requisitos:

Um servidor web é exigido para enviar as páginas geradas ao seu navegador web.

PHP é exigido para executar o software.

Um servidor de banco de dados é necessário para armazenar as páginas e os dados do site.
